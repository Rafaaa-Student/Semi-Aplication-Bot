# Website Dashboard Bot Raffasya VII

Website sederhana untuk memantau dan berinteraksi dengan data Bot Raffasya VII.

## Fitur
- **Home**: Statistik total users, poin hijau, dan buku.
- **Leaderboard**: Ranking poin hijau users.
- **Books**: Lihat koleksi buku dari database.
- **Search Books**: Cari buku berdasarkan keyword.

## Cara Menjalankan
1. Install dependencies: `pip install -r requirements.txt`
2. Jalankan: `python app.py`
3. Buka browser ke `http://127.0.0.1:5000/`

## Catatan
- Website ini membaca data dari file JSON yang sama dengan bot Discord.
- Pastikan file `poin_hijau.json` dan `database_buku_log.json` ada di folder induk.
